# 豆匣 DouXia

[English](README.md)

![豆匣 Logo](https://raw.githubusercontent.com/KylinLabAI/DouXia-App/master/resources/logo_256.png)

将散落的 AI 对话整理为有序、可搜索、可导出知识的桌面应用。

## 功能特性

- **内嵌 AI 对话 + 自动捕获** — 在豆匣中直接使用豆包、元宝、DeepSeek 或 ChatGPT，每条对话实时自动捕获并本地存储
- **统一对话管理** — 在一处浏览、全文搜索（FTS5）与筛选所有 AI 对话，不受来源平台限制
- **智能整理** — AI 自动打标签、话题聚类、知识图谱可视化及自定义标签管理
- **AI 知识抽取** — 生成精炼摘要、结构化知识卡片，并智能推荐清理低价值对话
- **灵活导出与分享** — 一键导出为 Markdown、HTML 或 PDF，支持局域网分享及数据库备份/恢复
- **离线可用** — 所有数据保存在本机；搜索、浏览、导出无需联网

## 快速开始

1. 下载对应平台的安装包
2. 运行安装程序并按照提示操作
3. 启动豆匣
4. 选择偏好的 AI 平台，在内嵌浏览器中登录
5. 开始对话——默认自动捕获对话内容
